  * н•ңмһҘмңјлЎң лҒқлӮҙлҠ” л№„мҰҲлӢҲмҠӨ лӘЁлҚё 100(кіӨлҸ„ лҚ°м“°лЎң): л№„мҰҲлӢҲмҠӨ лӘЁлҚё нҢҢм•…
  * лҚ°мқҙн„° л¬ён•ҙл Ҙ(м№ҙмӢңмҷҖкё° мҡ”мӢңнӮӨ): лҚ°мқҙн„° н•ҙм„қ лҠҘл Ҙ
  * лҰ° л¶„м„қ(м•ЁлҰ¬мҠӨн…Ңм–ҙ нҒ¬лЎӨ): м§Җн‘ң л¶„м„қ кіөл¶Җ
  * м„ём»Ёл“ң нҺӯк·„(мһ„мҠ№нҳ„): м§Җн‘ң л¶„м„қ кіөл¶Җ
  * л…јлҰ¬мқҳ кё°мҲ (л°”л°”лқј лҜјнҶ ): л…јлҰ¬л Ҙ кіөл¶Җ
  * лҚ°мқҙн„° л¶„м„қмқҳ нһҳ(мқҙнҶ  кі мқҙм№ҳлЎң): л¶„м„қл Ҙ кіөл¶Җ
  * лҚ°мқҙн„° кіјн•ҷмқ„ мң„н•ң нҶөкі„(м•Өл“ңлЈЁ лёҢлЈЁмҠӨ): лҚ°мқҙн„° мӮ¬мқҙм–ёмҠӨ нҶөкі„н•ҷ
* ML
  * вң… нҳјкіөлЁёмӢ (л°•н•ҙм„ ): мқёкіөм§ҖлҠҘ лЁёмӢ лҹ¬лӢқ л”Ҙлҹ¬лӢқ
  * вң… л”Ҙлҹ¬лӢқмңјлЎң кұ·лҠ” мӢңкі„м—ҙ мҳҲмёЎ(мңӨмҳҒм„ ): м•„нӮӨн…ҚнҠё, л”Ҙлҹ¬лӢқ н”„лЎңм қнҠё мӢӨм „ мҳҲм ң
  * л”Ҙл ӨлӢқмқ„ мқҙмҡ©н•ң м •нҳ• лҚ°мқҙн„° л¶„м„қ(л§ҲнҒ¬ лқјмқҙм–ё): лҚ°мқҙн„° мӮ¬мқҙм–ёнӢ°мҠӨнҠё, л”Ҙлҹ¬лӢқ мӢӨм „ мҳҲм ңмҷҖ лӘЁлҚё л°°нҸ¬
  * вң… нҢҢмқҙнҶ м№ҳ мІ«кұёмқҢ(мөңкұҙнҳё): нҢҢмқҙнҶ м№ҳм—җм„ң нҷңмҡ©
* Algorithm
  * вң… нҢҢмқҙмҚ¬ мҪ”л”© лҸ„мһҘ(лӮЁмһ¬мңӨ): нҢҢмқҙмҚ¬ AtoZ (+нҢҢмқҙмҚЁлӢү л§ӣліҙкё°)
  * мқҙкІғмқҙ м·Ём—…мқ„ мң„н•ң мҪ”л”© н…ҢмҠӨнҠёлӢӨ with нҢҢмқҙмҚ¬(лӮҳлҸҷл№Ҳ): м•Ңкі лҰ¬мҰҳ м •м„қ
  * н—Өл“ң нҚјмҠӨнҠё л””мһҗмқё нҢЁн„ҙ(м—җлҰӯ н”„лҰ¬лЁј): OOP л””мһҗмқё нҢЁн„ҙ
