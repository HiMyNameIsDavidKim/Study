# On-device AI 구현을 위한 언어모델 경량화 (SKT)

## 개요
* 디바이스: 공기청정기 로봇(SK인텔릭스의 NAMUH X A1)
* 음성 -> 펑션콜
* 인풋: 음성 (사투리, 발음, 소음, 노이즈)
* 아웃풋: 펑션콜
* A100 인스턴스 제공

## 평가 핵심
* 펑션콜 정확도 (50개, 90점)
* 응답시간 제한 (0.45초, 패널티 왕 큼)
* 메모리 제한 (3GB 이하, 런타임 모니터링)
* 데이터 증강, 합성 필요 (자유)
* (제공하는 GPU 인스턴스보다 좋은 모델로 합성하고 싶으면 알아서 합성해오면 됨)

## 참고
* 멀티턴 없음
* 펑션콜 목록 제공함
* LLM, STT는 베이스 모델 제공
* STT 모델 튜닝해도 됨
* 예외 처리 로직 자유롭게 구성해도 됨 (if문 알고리즘도 가능)
